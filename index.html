✨ 什麼是 Cosmos？

Cosmos 是一個革命性的 JavaScript 框架，旨在讓程式碼編寫變得如同玩積木般直觀、充滿樂趣與詩意。我們相信，複雜的開發過程應該被簡化為一系列可組合、易於理解的「積木」，讓您專注於創新，而非繁瑣的語法細節。

  • 直觀組合： 每個功能都是獨立的積木，隨心所欲地拼接。
  • 零學習負擔： API 設計極度簡潔，一看就懂，助您快速上手。
  • 即時生效： 堆疊積木即刻見效，享受即時回饋的開發樂趣。
  • 智能連動： 積木之間自動協同工作，簡化複雜邏輯。

🚀 快速開始

只需簡單幾步，即可開始您的 Cosmos 之旅。

步驟 1：引入 Cosmos

在您的 HTML 檔案中引入 Cosmos 框架： 

<script src="dist/cosmos.min.js"></script>
<div id="cosmos-app"></div>

步驟 2：堆疊您的第一個積木

使用 Cosmos 建立一個簡單的應用： 

new Cosmos.App()
    .title('我的第一個 Cosmos 應用')
    .text('用積木寫程式真的很有趣！')
    .button('點我看看')
    .onClick('你好，Cosmos！')
    .start();
