📖 API 文檔

Cosmos 框架的所有「積木」功能詳解。

1. UI 元件積木

.title()

設置應用程式的主標題。

語法:

.title('標題文字')

參數規則:

參數 類型 說明
標題文字 String 必要參數，顯示在頁面上的標題內容。

使用範例:

new Cosmos.App()
    .title('聯絡我們')
    .start();

.text()

顯示一段靜態文字。

語法:

.text('內容')

參數規則:

參數 類型 說明
內容 String 必要參數，要顯示的文字內容。支援 HTML 標籤。

使用範例:

new Cosmos.App()
    .text('歡迎使用 Cosmos 框架！')
    .text('支援 HTML: <strong>粗體文字</strong>。')
    .start();

.input()

建立一個文字輸入框。

語法:

.input('提示文字')

參數規則:

參數 類型 說明
提示文字 String 必要參數，輸入框的 placeholder 文字。

使用範例:

new Cosmos.App()
    .input('您的姓名')
    .input('您的電子郵件')
    .start();

.button()

建立一個可點擊的按鈕。

語法:

.button('按鈕文字')

參數規則:

參數 類型 說明
按鈕文字 String 必要參數，顯示在按鈕上的文字。

使用範例:

new Cosmos.App()
    .button('提交表單')
    .onClick('表單已提交！')
    .start();

.image()

顯示一張圖片，支援靈活尺寸設定與響應式特性。

語法:

.image('網址/路徑', 寬度?, 高度?, 'alt文字?')

參數規則:

參數 類型 說明
網址/路徑 String 必要參數，圖片的 URL 或本地路徑。
寬度 Number | undefined 可選參數，圖片的顯示寬度（像素）。若提供，則高度也應提供。
高度 Number | 'auto' | undefined 可選參數，圖片的顯示高度（像素）或 'auto' 實現自適應高度。
alt文字 String | undefined 可選參數，用於無障礙訪問的替代文字。

特殊功能:

  • 自適應支援：可使用 'auto' 作為高度參數，使圖片高度自適應寬度。
  • 參數邏輯：如果使用 寬度，就必須有 高度 參數（可以是數字或 'auto'）。

響應式特性:

  • 所有圖片都具備 max-width: 100% 響應式保護。
  • 固定尺寸圖片在小螢幕上會自動縮放，不會被切除。
  • 使用 object-fit: contain 保持圖片比例。
  • 原始尺寸圖片完全響應式適應容器寬度。

使用範例:

// 原始尺寸（完全響應式）
new Cosmos.App().image('https://placehold.co/600x200/4285f4/ffffff?text=Cosmos+Demo').start();

// 固定尺寸
new Cosmos.App().image('https://placehold.co/400x300/4285f4/ffffff?text=Fixed+Size', 400, 300, '固定尺寸圖片').start();

// 寬度固定，高度自適應
new Cosmos.App().image('https://placehold.co/400xauto/4285f4/ffffff?text=Auto+Height', 400, 'auto', '寬高自適應圖片').start();

.select()

建立一個下拉式選單。

語法:

.select(['選項A', '選項B', ...])

參數規則:

參數 類型 說明
選項陣列 Array<String> 必要參數，包含所有選項文字的陣列。

使用範例:

new Cosmos.App()
    .select(['技術支援', '業務合作', '其他'])
    .start();

.checkbox()

建立一個勾選框。

語法:

.checkbox('標籤文字')

參數規則:

參數 類型 說明
標籤文字 String 必要參數，勾選框旁邊顯示的標籤文字。

使用範例:

new Cosmos.App()
    .checkbox('訂閱我們的電子報')
    .start();

.slider()

建立一個可拖動的滑桿。

語法:

.slider(最小值, 最大值, 預設值?)

參數規則:

參數 類型 說明
最小值 Number 必要參數，滑桿的最小值。
最大值 Number 必要參數，滑桿的最大值。
預設值 Number | undefined 可選參數，滑桿的初始值。若未提供，則使用最小值。

使用範例:

new Cosmos.App()
    .slider(0, 100, 50) // 建立一個從 0 到 100，預設值為 50 的滑桿
    .start();

2. 樣式與佈局積木

.layout()

改變後續元素的排列方式（水平或垂直）。

語法:

.layout('row' | 'col')

參數規則:

參數 類型 說明
方向 String 必要參數，可以是 'row' (水平排列) 或 'col' (垂直排列)。

使用範例:

new Cosmos.App()
    .layout('row')
    .button('按鈕 1')
    .button('按鈕 2') // 按鈕將水平並排
    .layout('col')
    .text('文字 1')
    .text('文字 2') // 文字將垂直排列
    .start();

.color()

改變後續元素的預設顏色。

語法:

.color('顏色名')

參數規則:

參數 類型 說明
顏色名 String 必要參數，預設支援 'blue', 'green', 'red', 'purple', 'orange', 'pink'。

使用範例:

new Cosmos.App()
    .color('red')
    .button('紅色按鈕')
    .color('blue')
    .text('藍色文字')
    .start();

.animate()

為前一個被建立的元素加上動畫效果。

語法:

.animate('動畫名')

參數規則:

參數 類型 說明
動畫名 String 必要參數，支援 'fadeIn', 'slideIn', 'bounce', 'pulse'。

使用範例:

new Cosmos.App()
    .button('點我彈跳').animate('bounce')
    .text('漸入文字').animate('fadeIn')
    .start();

3. 表單驗證積木

為 input 或 select 元素添加多種驗證規則，會在使用者輸入時即時顯示錯誤訊息。

.required()

設定為必填欄位。

語法:

.required('錯誤訊息')

參數規則:

參數 類型 說明
錯誤訊息 String 必要參數，當欄位為空時顯示的錯誤提示。

使用範例:

new Cosmos.App()
    .input('用戶名').required('用戶名為必填')
    .start();

.email()

驗證 Email 格式。

語法:

.email('錯誤訊息')

參數規則:

參數 類型 說明
錯誤訊息 String 必要參數，當 Email 格式不正確時顯示的錯誤提示。

使用範例:

new Cosmos.App()
    .input('Email').email('請輸入有效的 Email 地址')
    .start();

.minLength()

驗證最小長度。

語法:

.minLength(長度, '錯誤訊息')

參數規則:

參數 類型 說明
長度 Number 必要參數，最小允許的字元數。
錯誤訊息 String 必要參數，當長度不足時顯示的錯誤提示。

使用範例:

new Cosmos.App()
    .input('密碼').minLength(6, '密碼至少需要 6 個字元')
    .start();

.maxLength()

驗證最大長度。

語法:

.maxLength(長度, '錯誤訊息')

參數規則:

參數 類型 說明
長度 Number 必要參數，最大允許的字元數。
錯誤訊息 String 必要參數，當長度超過時顯示的錯誤提示。

使用範例:

new Cosmos.App()
    .input('簡介').maxLength(100, '簡介不能超過 100 個字元')
    .start();

.numeric()

驗證是否為數字格式。

語法:

.numeric('錯誤訊息')

參數規則:

參數 類型 說明
錯誤訊息 String 必要參數，當輸入不是數字時顯示的錯誤提示。

使用範例:

new Cosmos.App()
    .input('年齡').numeric('請輸入數字')
    .start();

.pattern()

使用正則表達式自訂格式驗證。

語法:

.pattern(正則表達式, '錯誤訊息')

參數規則:

參數 類型 說明
正則表達式 RegExp 必要參數，用於驗證輸入格式的 RegExp 物件。
錯誤訊息 String 必要參數，當格式不符合時顯示的錯誤提示。

使用範例:

new Cosmos.App()
    .input('郵政編碼').pattern(/^\d{5}$/, '請輸入 5 位數字的郵政編碼')
    .start();

.onValid()

所有驗證通過後，點擊按鈕時觸發的動作。

語法:

.onValid(action)

參數規則:

參數 類型 說明
action String | Function 必要參數，可以是訊息字串或接收表單值的函式 function(values){...}。

使用範例:

new Cosmos.App()
    .input('用戶名').required('用戶名為必填')
    .button('註冊')
    .onValid('太棒了，所有資料都正確！')
    .start();

.onInvalid()

任何驗證失敗後，點擊按鈕時觸發的動作。

語法:

.onInvalid(action)

參數規則:

參數 類型 說明
action String | Function 必要參數，可以是訊息字串或接收表單值的函式 function(values){...}。

使用範例:

new Cosmos.App()
    .input('用戶名').required('用戶名為必填')
    .button('註冊')
    .onInvalid('請修正表單中紅色的錯誤提示')
    .start();

4. 事件處理積木

定義當特定事件發生時要執行的動作。

.onClick() / .buttonClick()

綁定在最後一個按鈕上，在點擊時觸發。

語法:

.onClick(action)

參數規則:

參數 類型 說明
action String | Function 必要參數，可以是訊息字串或接收所有表單值的函式 function(values){...}。

使用範例:

new Cosmos.App()
    .input('您的名字')
    .button('打招呼')
    .onClick(function(values) {
        alert(`你好, ${values['您的名字']}!`);
    })
    .start();

.onChange()

綁定在最後一個輸入元件上 (input, select, checkbox, slider)，在值變更時觸發。

語法:

.onChange(action)

參數規則:

參數 類型 說明
action String | Function 必要參數，可以是訊息字串或接收該元件新值的函式 function(value){...}。

使用範例:

new Cosmos.App()
    .input('您的名字').onChange(function(name) {
        console.log(`名字變成了: ${name}`);
    })
    .start();

5. 條件邏輯積木

取代傳統的 if/else，讓邏輯鏈更清晰。邏輯會在綁定的按鈕被點擊時觸發。

.when()

開始一個條件判斷。

語法:

.when(欄位名稱, 運算子, 值)

參數規則:

參數 類型 說明
欄位名稱 String 必要參數，要判斷的輸入欄位名稱。
運算子 String 必要參數，支援 '==', '!=', '>', '<', '>=', '<=', 'includes', 'empty'。
值 Any 必要參數，用於比較的值。

使用範例:

new Cosmos.App()
    .input('年齡').numeric('請輸入數字')
    .button('提交')
    .when('年齡', '>=', 18).then('您是成年人')
    .start();

.and()

附加一個「且」條件到前一個 .when() 或 .and() 判斷。

語法:

.and(欄位名稱, 運算子, 值)

參數規則:

參數 類型 說明
欄位名稱 String 必要參數，要判斷的輸入欄位名稱。
運算子 String 必要參數，支援的運算子與 .when() 相同。
值 Any 必要參數，用於比較的值。

使用範例:

new Cosmos.App()
    .input('分數')
    .button('檢查')
    .when('分數', '>=', 60).and('分數', '<=', 100).then('及格')
    .start();

.or()

附加一個「或」條件到前一個 .when() 或 .and() 判斷。

語法:

.or(欄位名稱, 運算子, 值)

參數規則:

參數 類型 說明
欄位名稱 String 必要參數，要判斷的輸入欄位名稱。
運算子 String 必要參數，支援的運算子與 .when() 相同。
值 Any 必要參數，用於比較的值。

使用範例:

new Cosmos.App()
    .input('偏好顏色')
    .button('判斷')
    .when('偏好顏色', '==', '紅').or('偏好顏色', '==', '藍').then('您喜歡原色')
    .start();

.then()

當前一個條件判斷成立時執行的動作。

語法:

.then(action)

參數規則:

參數 類型 說明
action String | Function 必要參數，可以是訊息字串或函式。

使用範例:

new Cosmos.App()
    .input('年齡').numeric('請輸入數字')
    .button('提交')
    .when('年齡', '>=', 18).then('您是成年人')
    .start();

.else()

當前一個 .when() 條件不成立時執行的動作。

語法:

.else(action)

參數規則:

參數 類型 說明
action String | Function 必要參數，可以是訊息字串或函式。

使用範例:

new Cosmos.App()
    .input('年齡').numeric('請輸入數字')
    .button('提交')
    .when('年齡', '>=', 18).then('您是成年人').else('您是未成年人')
    .start();

.otherwise()

所有 .when() 條件都不成立時執行的動作。

語法:

.otherwise(action)

參數規則:

參數 類型 說明
action String | Function 必要參數，可以是訊息字串或函式。

使用範例:

new Cosmos.App()
    .input('年齡').numeric('請輸入數字')
    .button('提交')
    .when('年齡', '>=', 18).then('您是成年人')
    .when('年齡', '<', 0).then('年齡不能為負數')
    .otherwise('請輸入有效的年齡')
    .start();

6. API 通訊積木

Cosmos 提供了完整的 API 通訊功能，讓您能夠輕鬆與後端服務進行互動。

.GET()

發送 GET 請求到指定 URL，用於獲取資料。

語法:

.GET(url)

參數規則:

參數 類型 說明
url String 必要參數，請求的目標 URL。

使用範例:

new Cosmos.App()
    .GET('https://jsonplaceholder.typicode.com/users/1')
    .onSuccess('歡迎 {name}！您來自 {address.city}')
    .onError('載入失敗：{error}')
    .start();

.POST()

發送 POST 請求到指定 URL，用於創建新資料。如果不提供 data 參數，會自動收集表單資料。

語法:

.POST(url, data?)

參數規則:

參數 類型 說明
url String 必要參數，請求的目標 URL。
data Object | undefined 可選參數，要發送的資料物件。如果省略，Cosmos 會自動收集當前容器中輸入元素的值。

使用範例:

// 自動收集表單資料
Cosmos.create('form-app')
    .input('請輸入標題')
    .input('請輸入內容')
    .button('發布')
    .onClick(() => {
        new Cosmos.App()
            .POST('https://jsonplaceholder.typicode.com/posts')
            .onSuccess('文章發布成功！ID: {id}')
            .onError('發布失敗：{error}');
    })
    .start();

// 手動提供資料
new Cosmos.App()
    .POST('https://api.example.com/users', {
        name: '張三',
        email: 'zhang@example.com'
    })
    .onSuccess('用戶 {name} 創建成功！')
    .start();

.PUT()

發送 PUT 請求到指定 URL，用於更新現有資料。如果不提供 data 參數，會自動收集表單資料。

語法:

.PUT(url, data?)

參數規則:

參數 類型 說明
url String 必要參數，請求的目標 URL。
data Object | undefined 可選參數，要發送的資料物件。如果省略，Cosmos 會自動收集當前容器中輸入元素的值。

使用範例:

new Cosmos.App()
    .PUT('https://api.example.com/users/123', {
        name: '新名稱',
        email: 'new@example.com'
    })
    .onSuccess('用戶資料更新成功')
    .onError('更新失敗：{error}')
    .start();

.DELETE()

發送 DELETE 請求到指定 URL，用於刪除資料。

語法:

.DELETE(url)

參數規則:

參數 類型 說明
url String 必要參數，請求的目標 URL。

使用範例:

new Cosmos.App()
    .DELETE('https://api.example.com/users/123')
    .onSuccess('用戶刪除成功')
    .onError('刪除失敗：{error}')
    .start();

.onSuccess()

定義成功回應的處理方式。支援字串模板、函數和目標元素格式。

語法:

.onSuccess(handler)

參數規則:

參數 類型 說明
handler String | Function | Object 必要參數。可以是字串模板 ('{key}'), 接收資料的函數 ((data) => {...}), 或包含 target 和 template 的物件 ({ target: 'id', template: '...' })。

使用範例:

// 字串模板格式
.onSuccess('歡迎 {name}！您的郵箱是 {email}')

// 函數格式
.onSuccess((data) => {
    console.log('收到資料：', data);
    // ... 複雜邏輯
})

// 目標元素格式
.onSuccess({
    target: 'result-div',
    template: '載入完成！用戶：{name}'
})

.onError()

定義錯誤回應的處理方式。支援字串模板、函數和目標元素格式。

語法:

.onError(handler)

參數規則:

參數 類型 說明
handler String | Function | Object 必要參數。可以是字串模板 ('{error}'), 接收錯誤的函數 ((error) => {...}), 或包含 target 和 template 的物件。

使用範例:

// 字串模板格式
.onError('請求失敗：{error}')

// 函數格式
.onError((error) => {
    console.error('API 錯誤：', error);
    // ... 複雜錯誤處理
})

.onLoading()

定義載入狀態的處理方式，在 API 請求開始時觸發。

語法:

.onLoading(handler)

參數規則:

參數 類型 說明
handler String | Function 必要參數。可以是訊息字串，或在載入時執行的函數。

使用範例:

// 字串格式
.onLoading('正在載入資料，請稍候...')

// 函數格式
.onLoading(() => {
    document.getElementById('spinner').style.display = 'block';
})
